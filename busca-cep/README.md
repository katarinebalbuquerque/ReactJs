# Busca CEP

<img src="image.png" width="310"/><br/>

### I. Sobre o projeto

* Consulta CEP usando a BrasilAPI.

* Interface simples com campo e botão de busca.

* Exibe logradouro, bairro, cidade e estado.

* Busca dinâmica e rápida sem recarregar a página.
